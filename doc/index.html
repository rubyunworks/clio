Clio is an advanced library for developing command line intefaces in the Ruby programming lanaguage. Clio includes an advances commandline parser, a quick base-class commandline parser, easy methods for getting use input and a console String class that makes it suprisingly easy to output beautiful console text.

The advanced parser can be used in a number of ways. The highest-level approach involves specifiying ones commandline in a help text like string.

      require 'clio'

      use = Clio::Usage.new %{
        --verbose -v          do it loudly

        document              generate documents
          --output=PATH -o    where to put it
          *FILES
      }

This syntax can be broken down further into the underlying ... 

      use = Clio::Usage.new
      use['--verbose -v']
      use['document']['--output=PATH -o']['*FILES']

      use.help(
        'document', 'generate docs',
        '--verbose', 'do it loudly'
      )

      use['document'].help(
        'output', 'where to put it'
      )

This then can be broken down fruther to the underlying DSL that Clio uses to create all usage specifications.

      use = Clio::Usage.new do
        command :document do
          help 'generate docs'
          option :output, :o do
            type 'PATH'
            help 'where to put it' 
          end
          argument 'FILES' do
            multiple
          end
        end
        option(:verbose, :v) do
          help 'do it loudly'
        end
      end

Once specified, it's time to parse and utilize the results.

      cli = use.parse('document -v -o doc/ README [A-Z]*')

      cli.command    #=> "document"
      cli.options    #=> {:output => "doc/", :verbose=>true}
      cli.arguments  #=> ["README", "[A-Z]*"]

      cli.verbose?   #=> true

What to output some really nice help text. Let's use Clio::String to spruce it up.

      cli.parse('--help')

      if cli.help?
        puts Clio::String.new(cli.help){ |s|
          s.gsub!(/^\w+\:/){ |w| w.bold.underline }
          s.gsub!(/[-]{1,2}\w+/){ |w| w.blue }
        }
      end
